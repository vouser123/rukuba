Form Parameters & Modifiers

When to use Instructions vs. Parameters:
  • Instructions: If location/setup is always the same (e.g., "Place hands on hips", "Stand on foam pad")
  • Parameters: If it varies and needs to be logged each session (e.g., band color/resistance, step height, ball size)
Common form modifiers:
  • Alternating: Specify in instructions if sides alternate within each set
  • Duration vs. Hold: Duration (sec) = continuous movement time; Hold (sec) = static hold time
  • Tempo: Specify as "3-1-3" (3s down, 1s pause, 3s up) in dosage notes
  • Range: If range varies (e.g., "to tolerance", "partial"), note in dosage notes

Sets/Reps for Sided Exercises

  • Bilateral (both sides same set): Enter as "3 sets x 10 reps" - patient logs once
  • Unilateral (each side separate): Enter as "3 sets x 10 reps per side" in dosage notes - patient logs each side separately
  • Alternating: Specify in instructions (e.g., "Alternate legs each rep")
  • Example: "Single leg balance" ‚Üí "3 sets x 30s per side" ‚Üí patient logs L and R separately
